# CacheBash Documentation

CacheBash is the coordination layer between your AI coding agents and your phone. Connect any MCP-compatible client — Claude Code, Cursor, VS Code, Gemini CLI — and your agents can create tasks, exchange messages, track sessions, and ask you questions from anywhere.
